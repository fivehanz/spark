# Apache Spark

### Download Spark

Date: March 2021
spark release 3.1.1 
package type "prebuilt for Hadoop 2.7"
openjdk 15.0.2
python 3.9.2

### PySpark Shell

To start PySpark on Linux/OS X

```
$HOME/spark-3.1.1-bin-hadoop2.7/bin/pyspark
```

or install with pip and start pyspark

```
pip install pyspark
pyspark
```
